# Light Switch v2.0

## Usage

### Run in development mode

Running in development mode will re-compile and hot-reload on changes.

```bash
yarn dev
```

### Run in production mode

```bash
yarn build
yarn serve
```

### Run tests

```bash
yarn test
```

In watch mode:

```bash
yarn test-watch
```

### Run type checking

```bash
yarn typecheck
```

### Run linting

```
yarn lint
```

### Build package

```
yarn dist
```
